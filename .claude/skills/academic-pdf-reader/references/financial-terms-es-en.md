# Financial Terms: Spanish-English Reference

## Purpose

This reference provides comprehensive Spanish-English translations for financial terminology commonly encountered in MBA finance coursework, particularly in the Dirección de Finanzas course materials.

Use this guide when:
- Translating financial concepts between Spanish and English
- Clarifying terminology in bilingual contexts
- Understanding course materials written in Spanish
- Communicating financial analysis to English-speaking audiences

## Chilean-Specific Conventions

### Number Formatting

**Chilean Format:**
- Thousands separator: period (.)
- Decimal separator: comma (,)
- Example: 1.234.567,89

**US/International Format:**
- Thousands separator: comma (,)
- Decimal separator: period (.)
- Example: 1,234,567.89

**Conversion Notes:**
When extracting from PDFs, pay attention to:
- Chilean PDFs use: "12,5%" (12.5%)
- US PDFs use: "12.5%" (12.5%)
- Large numbers: "1.234" (Chilean) = "1,234" (US)

## Usage Guidelines

### When to Translate

**Translate to English when:**
- User explicitly requests English terminology
- Communicating with English-speaking audience
- Writing for international publication
- Clarifying ambiguous terms

**Keep in Spanish when:**
- Analyzing course materials in Spanish
- Original terminology is clear
- User's context is Spanish-language
- Preserving authenticity of source

### Best Practices

1. **Preserve original terms in quotes**: "apalancamiento" (leverage)
2. **Provide both when helpful**: WACC (CPPC)
3. **Note regional variations**: "utilidad" vs "ganancia" (both mean profit)
4. **Respect number formatting**: Convert Chilean format to international when needed
5. **Context matters**: "rentabilidad" can mean return, profitability, or yield depending on context
